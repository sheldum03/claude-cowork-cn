Claude Desktop 中文补丁 · 轻量版

一个 Python 脚本为 Claude Desktop 添加简体/繁体中文界面，无需修改 app.asar，几秒钟完成安装。


GitHub 仓库

快速安装

macOS：

git clone https://github.com/sheldum03/claude-cowork-cn.git
cd claude-cowork-cn
sudo python3 install.py

Windows：

git clone https://github.com/sheldum03/claude-cowork-cn.git
cd claude-cowork-cn
python install.py

安装后 Claude Desktop 会自动重启并显示中文界面。

特性

轻量安全

不修改 app.asar、不重签名、不碰可执行文件，只复制翻译资源。

零依赖

纯 Python 标准库，单文件脚本，无需 pip install 任何东西。

一键还原

运行 --restore 即可完全恢复英文，或直接重装 Claude。

三种中文

支持简体中文、繁体中文（台湾）、繁体中文（香港）。

常见问题

Claude Desktop 怎么设置中文？
官方暂未内置中文。使用本工具运行一条命令即可添加中文界面，支持 macOS 和 Windows。
Claude Desktop 有没有中文版？
官方没有，但 claude-cowork-cn 可以为其添加约 70% 的中文界面覆盖。
安全吗？会不会破坏 Claude？
不会。本工具只做文件复制和配置写入，不碰核心代码。随时可用 --restore 还原。
Claude 更新后中文消失了？
更新会覆盖翻译文件，重新运行一次安装脚本即可恢复。
和完整版有什么区别？
完整版（javaht/claude-desktop-zh-cn）修改 app.asar 实现接近 100% 覆盖率。本轻量版不碰 app.asar，覆盖率约 70%，安装更简单、风险更低。

工作原理

install.py 是一个无第三方依赖的单文件 Python 脚本，做四件事：

1. 定位 Claude Desktop 安装目录（自动识别 macOS/Windows）
2. 把中文翻译 JSON 与 Claude 自带的 en-US.json 按 key 合并，未翻译的回退英文
3. 用正则在前端 JS bundle 的语言数组里追加中文代码
4. 把用户配置文件的 locale 设为所选语言
